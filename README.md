### Machine Learning Experiments

This repository contains practical machine learning experiments and ready-to-run code examples demonstrating the application of common algorithms and data analysis techniques. It is structured by topics and includes both supervised and unsupervised learning approaches, along with exploratory data analysis and visualization.

### 1. Supervised Learning
- Logistic Regression
- Linear Regression
- Support Vector Machines
- Decision Trees
- Random Forests

### 2. Unsupervised Learning
- K-Means Clustering
- Hierarchical Clustering (Linkage Criteria)
- Isolation Forest (Anomaly Detection)

### 3. Data Analysis & Visualization
- Exploratory Data Analysis (EDA)
- Data distribution plots
- Correlation metrics
- Feature insights

### Purpose
- Provide practical, ready-to-use machine learning examples
- Demonstrate model implementation, evaluation, and visualization
- Serve as a reference for applied machine learning techniques
